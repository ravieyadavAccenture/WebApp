How to open git bash:
	- Go to the file directory > Right click > Git Bash Here

Initialize
	- On git bash run (after $ sign is visible on the command line)
		git bash

Set git hub code as web app
	- On git hub repository > Settings > GitHub Pages
	- Change source to master branch
	- This will, in the GitHub Pages section, show a url, which is the web site url
		- https://ravieyadavaccenture.github.io/WebApp/

Commands:
	- git bash
		- Initializes git bash
	- git status
		- Checks status of various files in the folder where bash is running
	- git add .
		- All files will be added to git. Dot at the end is for all files and folders/subfolders to add.
	- git commit -m "comment while commiting" .
		- Commits the files added. Dot at the end is for all files and folders/subfolders to commit.
	- git remote add origin https://github.com/ravieyadavAccenture/WebApp.git
		- Remote origin will be added to local git repository
	- git push -u origin master
		- Push the code into the repository
